Nous avons simulé le fonctionnement d’une banque en utilisant trois tableaux principaux comme base de données :

-Le premier tableau est celui des clients, dans lequel sont enregistrées les informations personnelles telles que l’ID du client,
son nom complet, son e-mail et son numéro de téléphone.

-Le deuxième tableau concerne les comptes bancaires ; il contient pour chaque compte un identifiant unique, le numéro du compte, 
le type de compte (par exemple compte courant ou épargne) ainsi que le solde disponible.

-Le troisième tableau est celui des transactions, qui permet de suivre l’ensemble des opérations effectuées. Chaque transaction enregistre un ID, 
l’ID du compte concerné, le type d’opération (dépôt ou retrait), le montant et la date de la transaction.
